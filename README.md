# 🌍 Analyse des Émissions de CO2 (1960-2019)

## 📌 Contexte du Projet
Ce projet a été réalisé dans le cadre d'un **Travaux Pratiques** de **Base de Données** en **L2 DSBD**.  
L'objectif est d'analyser les données d'émissions de CO2 par pays sur plusieurs décennies (1960 à 2019) afin d'en extraire des tendances, des indicateurs clés et des classifications pertinentes.

---

## 🎯 Objectifs
- Identifier les **années extrêmes** (début et fin de la série).
- Comparer les **émissions annuelles** par pays.
- Classer les **pays les plus polluants**.
- Mettre en évidence les **émissions excessives** (> 200 000 unités) depuis 2016.
- Calculer la **stabilité des émissions** via l'écart-type.
- Catégoriser les pays selon leur niveau d'émission.

---

## 🛠️ Technologies Utilisées
- **PostgreSQL** (ou MySQL)
- **SQL** (requêtes d'agrégation, sous-requêtes, vues)
